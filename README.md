# Create T3 App

## How to work locally

This project uses [Bun](https://bun.sh/) as a package manager.

```sudo npm install -g bun```

--- 
Steps:

- ```git clone git@github.com:svenrisse/E-Commerce-App.git```

- ```cd E-Commerce-App```

- ```bun install```

- ```bun run db:push```

- ```cp .env.example .env```

- Add values to .env

- ```bun run dev```
